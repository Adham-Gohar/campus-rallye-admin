## Installation

Install the application dependencies by running:

```sh
npm install

Development

Start the application in development mode by running:

sh

npm run dev

Entwicklungsmodus

Um die Webanwendung im Entwicklungsmodus einzurichten und auszuführen, folgen Sie diesen Schritten:

    Abhängigkeiten installieren:

    Stellen Sie sicher, dass alle benötigten Abhängigkeiten installiert sind, indem Sie den folgenden Befehl im Stammverzeichnis des Projekts ausführen:

    sh

npm install

Entwicklungsmodus starten:

Starten Sie die Anwendung im Entwicklungsmodus mit dem folgenden Befehl:

sh

    npm run dev

    Zugriff auf die Anwendung:

    Nachdem der Entwicklungsmodus gestartet wurde, können Sie auf die Anwendung über Ihren Webbrowser zugreifen. Standardmäßig ist die Anwendung unter http://localhost:3000 erreichbar.

    Fehlerbehebung:

    Falls beim Starten der Anwendung im Entwicklungsmodus Probleme auftreten, überprüfen Sie die Konsolenausgaben auf Fehlermeldungen und stellen Sie sicher, dass alle Abhängigkeiten korrekt installiert sind.

    Konfigurationsdateien:

    Stellen Sie sicher, dass alle notwendigen Konfigurationsdateien (wie .env) korrekt eingerichtet sind, um die Anwendung im Entwicklungsmodus auszuführen.

Mit diesen Schritten können Sie die Webanwendung im Entwicklungsmodus einrichten und ausführen. Bei weiteren Fragen oder Problemen wenden Sie sich bitte an das Entwicklungsteam.
